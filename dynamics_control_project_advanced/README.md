# MASSIVE Linux Desktop Dynamics & Control Project

Research-lab style project including:
- Mass-spring-damper
- Pendulum
- Cart-pole
- PID, LQR controllers
- Kalman Filter, EKF
- Full physics engine (Lagrangian)
- Simulation, 3D animations
- Matplotlib GUI sliders
- LaTeX PDF report generation
- Examples and tests
- Linux-ready
